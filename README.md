# Easy to learn — Digital Marketing Website

A single-page website for **Easy to learn**, a digital marketing education brand.

## Features

- Sticky navigation with About, Courses, and Get Started
- Hero, About, Courses, and Contact sections
- Responsive design (desktop, tablet, mobile)
- Contact form with client-side validation
- Smooth scrolling and active nav highlighting

## Project Structure

```
easy-to-learn/
├── index.html
├── css/styles.css
├── js/main.js
├── assets/logo.svg
└── README.md
```

## How to Preview

**Option 1:** Open `index.html` directly in your browser.

**Option 2:** Run a local server:

```powershell
cd C:\Users\Vijeth\Projects\easy-to-learn
python -m http.server 8080
```

Then visit [http://localhost:8080](http://localhost:8080).
